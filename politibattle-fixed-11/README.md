# ToolHub AI — 66 Free Online Tools + Multiplayer Quiz Platform

Production-ready SaaS web application with **66 fully functional tools**, a **real-time multiplayer quiz system** (Classic + Gold Quest modes), **user accounts with premium subscriptions**, a full **admin dashboard**, **Google SEO optimization**, and **AdSense monetization**.

## What's Included

### 66 Tools (all bug-fixed, server-side processing)

**26 Text Tools:** AI Content Detector, AI Text Humanizer, Paraphrasing Tool, Text Summarizer, Grammar Fixer, Word Counter, Case Converter, Tone Analyzer, Sentence Expander, Text Cleaner, Bullet Point Converter, Cliche Detector, Markdown Previewer, Emoji Picker, Lorem Ipsum Generator, Word Frequency Analyzer, Text Diff Checker, Readability Checker, Text to Speech, Hashtag Generator, Email Subject Generator, Plagiarism Highlighter, Sentence Rewriter, Active Voice Converter, Character Limit Checker, Markdown to HTML.

**12 Media Tools:** Image Converter, Image Resizer, YouTube Thumbnail Downloader, QR Code Generator, Color Palette Generator, Base64 Encoder/Decoder, Image to Text (OCR), Color Picker, Favicon Generator, GIF Maker, Screenshot Resizer, CSS Gradient Generator.

**28 Utility Tools:** Password Generator, Timestamp Converter, JSON Formatter, Random Name Generator, Unit Converter, Scientific Calculator, Binary Converter, URL Encoder/Decoder, IP Lookup, Hash Generator, Regex Tester, CSS/HTML/JS Minifier, Color Contrast Checker, Currency Converter, Age Calculator, Percentage Calculator, BMI Calculator, Time Zone Converter, UUID Generator, Number to Words, Random Picker, Slug Generator, Morse Code Translator, CSV to JSON, Pomodoro Timer, Typing Speed Test.

### Multiplayer Quiz System
- **Classic Mode** — Kahoot-style live questions, streaks, time bonuses, real-time leaderboard
- **Gold Quest Mode** — Individual async play, mystery chest RNG, gold economy
- **Socket.io** real-time with Redis or in-memory fallback
- **Teacher dashboard** — host games, view results, manage student accounts

### User System & Monetization
- Email + Google OAuth sign-in
- Free tier (10/day), Premium ($9.99/mo via PayPal)
- Google AdSense integration
- Full admin panel

### SEO & Google Rich Results
- Server-side rendered meta tags, Open Graph, Twitter Cards
- JSON-LD: WebSite, SoftwareApplication, FAQPage, BreadcrumbList, HowTo, ItemList, Quiz
- Dynamic sitemap.xml (cached, auto-generated)
- Smart robots.txt

### Dashboard Analytics
- Donut pie charts (category breakdown, top tools)
- 7-day activity bar chart
- Stats cards with real-time data

## Quick Start

```bash
cp .env.example .env
npm install
npm start
```

Default admin: `admin@toolhub.ai` / `Admin@123`

## Deploy to Render

Push to GitHub, connect in Render — auto-detects render.yaml.
